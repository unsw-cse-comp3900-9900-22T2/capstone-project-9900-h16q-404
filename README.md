# How to run application:

## Run backend

Install required backend libraries:

$ pip install -r backend_requirements.txt

Start backend Flask application:

$ flask run

## Run frontend

Navigate to frontend folder:

$ cd frontend

Install all required frontend libraries:

$ npm install

Run frontend UI:

$ npm run start
